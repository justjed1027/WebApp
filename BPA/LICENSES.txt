================================================================================
                        IMPORTANT NOTES
================================================================================

1. NO EXTERNAL CDN SCRIPTS
   SkillSwap intentionally avoids external JavaScript frameworks (jQuery,
   React, Vue, etc.) and CSS frameworks (Bootstrap, Tailwind). This ensures:
   - Faster page loads
   - Smaller bundle sizes
   - Complete control over styling
   - Easier maintenance
   - Reduced external dependencies

2. INLINE SVG ICONS
   All SVG icons are embedded directly in HTML/PHP code. This provides:
   - No external icon library dependencies
   - Full color customization via CSS
   - Fast loading
   - Improved accessibility

3. BACKEND-ONLY GPL
   MySQL (GPL v2) is used only for backend database operations. The GPL
   license does not apply to the SkillSwap frontend code, which uses
   standard web technologies (HTML, CSS, JavaScript) with no license
   restrictions.

4. GOOGLE FONTS COMPLIANCE
   Google Fonts are freely available and licensed under OFL. According to
   Google's official documentation, no code-level attribution is required,
   though attribution in documentation (like this file) is appreciated.

5. CUSTOM SVG ELEMENTS
   Some SVG icons are custom-created for SkillSwap and are not covered by
   external licenses.
