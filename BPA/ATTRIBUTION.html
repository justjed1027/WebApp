2. SVG Icons & Iconography

Bootstrap Icons (SVG Style)

SkillSwap uses SVG icons that follow the Bootstrap Icons design aesthetic. These are embedded directly as inline SVG elements in the HTML/PHP code. Icons are used extensively throughout the application for navigation, actions, and visual indicators.

Icon Classes & Usage

Icons Used:

  • bi-person-fill - User profile indicators
  • bi-person-fill-add - Add connection/user actions
  • bi-house-door-fill - Home/dashboard navigation
  • bi-chat-dots-fill - Messaging/discussion icons
  • bi-chat-dots - Chat/comment indicators
  • bi-file-text-fill - Document/note references
  • bi-people-fill - Group/community indicators
  • bi-book - Learning resources
  • bi-calendar - Date/event indicators
  • bi-calendar-event - Event management
  • bi-gear-fill - Settings/preferences
  • bi-box-arrow-right - Logout/exit actions
  • bi-brightness-high - Light theme toggle
  • bi-moon - Dark theme toggle
  • bi-search - Search functionality
  • bi-envelope - Email/notification indicators
  • Plus various chevrons, arrows, and status indicators

Files Using Icons: connections.php, courses.php, course-list.php, course-detail.php, calendar.php, events.php, post.php, forums.php, notifications.php, newpage/index.php, and many others.

MIT License

Bootstrap Icons are licensed under the MIT License. Free for use in commercial and personal projects.

Bootstrap Icons Official Site: https://icons.getbootstrap.com/

Custom SVG Elements

Description: Some SVG elements are custom-created or adapted specifically for the SkillSwap application design.

Examples: Dropdown arrows, custom status indicators, theme toggles

License: Proprietary (created for SkillSwap)

Custom SVG icons are the intellectual property of the SkillSwap development team.

✓ Implementation Note: All icons are embedded as inline SVG elements in the HTML/PHP code. This approach provides: (1) No external dependencies, (2) Full color customization via CSS, (3) Fast loading, and (4) Improved accessibility with proper ARIA labels.

✓ Minimalist Approach: SkillSwap intentionally uses minimal external dependencies. The application uses only:
  • Vanilla JavaScript (no jQuery, React, Vue, or other frameworks)
  • Custom CSS (no Bootstrap, Tailwind, or other CSS frameworks)
  • Standard browser APIs (LocalStorage, Fetch, etc.)
  • Only external dependencies: Google Fonts and Bootstrap Icons style

This approach provides: faster page loads, smaller bundle sizes, complete control over styling, and easier maintenance.
